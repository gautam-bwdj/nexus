# Nexus: Legends of Tomorrow - Capstone Project

A futuristic hero-based game website built with **HTML and CSS** for the SNW Capstone Project.

## 🎮 Project Overview

**Project Name:** Nexus  
**Theme:** Games (Hero-Based Combat)  
**Reference Inspiration:** Overwatch  
**Tech Stack:** HTML5 & CSS3 only

## 🌟 Features

### Design Highlights
- **Futuristic Cyber Aesthetic**: Neon blue (#00d4ff) primary color with dark theme
- **Custom Typography**: Orbitron for headings, Rajdhani for body text
- **Responsive Design**: Mobile-first approach with media queries
- **Modern CSS Features**: Flexbox, Grid, gradients, animations, and transitions
- **Accessibility**: Proper semantic HTML and color contrast ratios

### Pages
1. **Home (index.html)**
   - Hero section with animated title and background
   - Game statistics showcase
   - Features grid (6 game features)
   - Hero preview showcase
   - Call-to-action section
   
2. **Heroes (heroes.html)**
   - Filterable hero grid (24 heroes mentioned)
   - 6 detailed hero cards with:
     - Role badges (Assault, Tank, Duelist, Support)
     - Abilities
     - Stat bars (Damage, Mobility, Survivability)
   - Role information section

3. **Gameplay (gameplay.html)**
   - Core mechanics breakdown (4 sections)
   - Game modes (6 different modes)
   - Maps showcase with descriptions
   - Progression system
   - Pro tips section

4. **Gallery (gallery.html)**
   - Masonry-style image grid
   - Categories: Screenshots, Concept Art, Heroes, Environments
   - Downloadable wallpapers section

5. **About (about.html)**
   - Game lore and story
   - Vision and mission
   - Development timeline
   - Team section
   - Contact form

## 🎨 Design System

### Color Palette
- **Primary**: #00d4ff (Cyber Blue)
- **Secondary**: #8b5cf6 (Purple)
- **Accent**: #f59e0b (Amber)
- **Dark Backgrounds**: #0a0a0f, #121218, #1a1a24
- **Gradients**: Linear gradients for buttons and accents

### Typography Scale
- Display (Orbitron): 700-900 weight
- Body (Rajdhani): 300-700 weight
- Headings: 2rem - 5rem
- Body: 1rem - 1.25rem

### Components
- Animated navigation bar with hover effects
- Gradient buttons with glow effects
- Card components with hover animations
- Timeline with markers
- Progress bars
- Filter buttons
- Contact form

## ✨ CSS Features Used

- **CSS Grid & Flexbox**: Modern layout techniques
- **CSS Variables**: Custom properties for design tokens
- **CSS Animations**: Keyframe animations for smooth effects
- **CSS Transitions**: Hover states and interactive elements
- **CSS Gradients**: Linear and radial gradients
- **CSS Filters**: Drop shadows and glows
- **Media Queries**: Responsive breakpoints
- **Pseudo-elements**: ::before, ::after for decorative elements

## 📝 Key Design Decisions

1. **Color Choice**: Cyber blue (#00d4ff) was chosen for its futuristic feel and excellent contrast against dark backgrounds
2. **Typography**: Orbitron provides a tech/sci-fi feel perfect for a futuristic game
3. **Layout**: Grid-based layouts for modern, clean appearance
4. **Animations**: Subtle hover effects and entrance animations enhance user experience
5. **Imagery**: High-quality sci-fi hero images from Unsplash/Pexels

## 📱 Responsive Breakpoints

- **Desktop**: 1024px and above
- **Tablet**: 768px - 1023px
- **Mobile**: Below 768px
